## 毕业设计 - 农业温室大棚

快毕业了，是时候准备毕业设计了，这里记录了咱毕业设计的全部内容以及源代码

具体功能是是通过STM32与传感器与控制器联动，进行控制，并将数据传输到树莓派中的数据库存起来，并且对外提供一个WEB服务器和API，可以通过浏览器和客户端进行控制访问和联动
